#Double range slider jQuery plugin#

**doubleRangeSlider** is jQuery plugin for creating double range slider control

##Version##
0.1.0

##Usage##
#Horizontal#
	$(document).ready(function(){
   		$('#Slider1').doubleRangeSlider();
   		
    });
#vertical#
 	$(document).ready(function(){
 		$('#Slider2').doubleRangeSlider({
   			direction:'vertical'
   		});
   	});
